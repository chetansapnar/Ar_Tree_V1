AR Model Test — Teak Tree

This page loads the `teak_tree.glb` model from /models/teak_tree.glb. On mobile, use Chrome (Android) or Safari (iOS) to trigger AR features.

Buttons:

  • On desktop: rotate/zoom the model with your mouse.
  • On mobile: an AR button should appear in the viewer UI — tap it to enter AR.

If your browser downloads the file instead of rendering the model, open the site root (e.g. http://localhost:3000/) and then navigate to this page; the project dev server serves the static model files at /models/.
